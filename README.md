# git hired

> AI-powered dev job board with semantic matching

## Structure
- `frontend/` — Next.js 15 + TypeScript + Tailwind + shadcn/ui
- `backend/` — NestJS + Prisma + PostgreSQL (Supabase)

## Getting started

### Frontend
```bash
cd frontend
npm install
npm run dev
```

### Backend
```bash
cd backend
npm install
npm run start:dev
```
